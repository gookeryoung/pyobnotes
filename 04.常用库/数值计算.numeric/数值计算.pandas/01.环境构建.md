# pandas.环境构建

`pandas` 通常配套 `numpy、matplotlib、scipy、ipython` 等库使用，可按此进行安装：

```bash
pip install numpy scipy matplotlib ipython pandas
```

安装后进行测试：

```python
import pandas as pd

pd.__version__
```

输出结果：

```
1.4.2
```
