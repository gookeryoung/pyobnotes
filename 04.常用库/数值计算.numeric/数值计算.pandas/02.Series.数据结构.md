# Series.数据结构

`Series` 是一维数组对象，由一组 `数据（data）` 和与之对应的 `标签（index）` 共同组成，如下图：

![[pandas_series_structure.png]]

图中，`0～9` 为标签，`Lois…Diana` 为数据。

以常规字符串构造的 `Series` 为例：

```python
import pandas as pd

s1 = pd.Series(list('ABCD'))
```

将得到：

```python
0 A
1 B
2 C
3 D
length: 4, dtype: object
```

其中，`0，1，2，3` 就是标签，`A，B，C，D` 就是数据。
