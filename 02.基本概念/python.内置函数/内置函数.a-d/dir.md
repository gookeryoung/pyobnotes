# dir([object])

返回 `object` 对象的有效属性列表，没有参数时则返回本地作用域的名称列表。

## 概述

对于不同类型的对象，dir 返回不同的信息：

- 如果对象是模块，则返回包含模块的属性名称；
- 如果对象是类型或者类对象，则返回包含它们的属性名称，**并递归查找所有基类属性**；
- 否则，列表包含对象的属性名称，它的类属性名称，并且递归查找它的类的所有基类的属性。
