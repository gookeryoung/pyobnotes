# dict([mapping/iterable,] **kwarg)

创建一个新的字典，基于可选位置的参数和可能为空的关键字参数集进行初始化。

## 示例

```python
d1 = dict([('a', 95), ('b', 85), ('c', 90)])
# {dict:3}{'a':95, 'b':85, 'c':90}
d2 = dict(a=95, b=85, c=90)
# {dict:3}{'a':95, 'b':85, 'c':90}
```
