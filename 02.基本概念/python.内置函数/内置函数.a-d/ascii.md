# ascii

返回一个字符串，表示对象可打印形式，不对 `\x` , `\u` , `\U` 等字符进行转义。

## 示例

```python
s1 = '你好'
# {str: 2}'你好'
s2 = ascii(s1)
# {str: 14} "\u4f60\u597d"

# 查看每位，注意'\\'是一位字符，代表'\'
lis = [x for x in s2]
# {list: 14}["'", '\\', 'u', '4', 'f', '6', '0', '\\', 'u', '5', '9', '7', 'd', "'"]

s3 = '你好，中国'
# {str: 6}'你好，中国'
s4 = ascii(s3)
# {str: 38}"'\\u4f60\\u597d\\uff0c\\u4e2d\\u56fd'"
```
