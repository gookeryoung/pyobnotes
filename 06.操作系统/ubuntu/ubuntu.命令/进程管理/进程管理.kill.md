# 进程管理.kill

```bash
# 结束进程, 后接pid，需要用ps aux|grep [name]配合查询
kill 59935

# 强制结束进程
kill -9 59980

# 结束进程，后接进程名
pkill chrome

# 结束全部进程，后接进程名
killall -9 chrome
```
